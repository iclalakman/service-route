# service--route

* **Geospatial Coordinate Acquisition:** Designed a workflow pipeline that dynamically ingests the GPS coordinates (Latitude/Longitude pairs) of corporate employees to establish precise spatial nodes.
* **Shortest Traverse Path Calculation:** Implemented algorithmic solutions to solve the Traveling Salesperson Problem (TSP) and Capacitated Vehicle Routing Problem (CVRP), mapping coordinates onto a weighted graph to calculate the absolute shortest operational mutation path.
* **Operational Constraint Synchronization:** The engine dynamically routes the multi-passenger shuttle fleet, ensuring the generated sequence minimizes fuel metrics and transit latency while strictly adhering to shift timetables and vehicle occupancy constraints.
